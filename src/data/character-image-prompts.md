# Character Avatar Prompts — Nano Banana

Style brief: Dark, moody portrait illustration. Square format (1:1 aspect ratio). Warm charcoal background (#1a1a1d). Painterly digital art with visible brushwork, desaturated palette with a single terracotta/amber accent. Shoulders-up composition, slightly off-center. Dramatic chiaroscuro lighting from one side. Expression should convey the character's psychological state — unsettling but human. No text.

**AI assistant — the language model responding in conversation**

Two circular optical sensors glow with a faint terracotta-amber light where eyes would be, set deep in shadowed recesses.
